Meu primeiro arquivo para trabalhar com CSS, ensinado pelos professores Gui e Rafa. Sexta-feira 27/09/2019.

O CSS é responsável pela beleza dos sites, ele modifica as propriedades dos elementos da pagina.

CSS quer dizer Cascade Style Sheets, que são folhas de estilo de cascata, e funcionam basicamente selecionando um grupo de atributos e mudando suas propriedades.
